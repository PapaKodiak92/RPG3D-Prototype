# Lesson 02: Click-to-Move

## What we are building

The player will click on the ground and the character will walk toward that spot.

This is the first RuneScape-style movement feature.

---

## Coding ideas

This lesson teaches:

- Variables
- Functions
- Mouse input
- Raycasting
- 3D positions
- Movement speed

---

## Important vocabulary

### Variable

A named value the game remembers.

Example:

```gdscript
var move_speed = 4.0
```

### Function

A named set of instructions.

Example:

```gdscript
func move_player():
    pass
```

### Raycast

A way to shoot an invisible line from the camera into the 3D world to see what the player clicked.

---

## What to test

- Click near the player.
- Click far away.
- Change the movement speed.
- Try clicking somewhere that is not the ground.

---

## Kid challenge

Change the movement speed and answer:

- What number feels too slow?
- What number feels too fast?
- What number feels right?
